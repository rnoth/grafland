grafland

Grafland is a collection of unique; userland tools, cross compiling toolchains, math libraries, arbitrary precision math libraries, terminal rendering libraries, standard C libraries, an irc client, a shell and a cross build system. It also contains a window manager, editor, system monitor and other useful tools and shell scripts.
