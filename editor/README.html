grafland editor




DESCRIPTION:

	A simple text editor with no dependencies



CONTROLS:

	ctrl-x to save
	ctrl-n to quit



DEPENDENCIES:

	Editor does not depend on ncurses/curses.

	Editor's screen drawing library is located
	at grafland's ../lib/hexen.h. Which is 
	a small (<1000loc) ANSI screen drawing 
	library.
