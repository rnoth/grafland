NAME:

	Cbin

	

DESCRIPTION:

	Cbin is a set of userland utilities which strives for POSIX
	compliance. When a feature is implemented it is to meet POSIX
	specifications. No additional features are added.


BUILDING:

	cd cbin
	make



INSTALLATION:

	cd cbin
	make install DESTDIR=/path/to/install/
