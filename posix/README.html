grafland userland utilities (bin)
	

DESCRIPTION:

	grafland's bin is a set of userland utilities which strives for POSIX
	compliance. When a feature is implemented it is to meet POSIX
	specifications. No additional features are added.


BUILDING:

	cd bin
	make



INSTALLATION:

	cd bin
	make install DESTDIR=/path/to/install/
